условие: 
Implement function ToLowerCase() that has a string parameter str, and returns the same string in lowercase.

 

Пример:

Input: "Hello"
Output: "hello"

Решение:

/**
 * @param {string} str
 * @return {string}
 */



var toLowerCase = function(str) {
  var res = str.toLowerCase();
  return res;
}
